Routing Massal

Tool Pengukur Antara Lokasi A dengan Lokasi B Secara Masal

Hitung jarak jalan aktual dari banyak titik ke alat produksi terdekat dengan bantuan OSRM dan data jalan OpenStreetMap. Halaman ini kini dibungkus dalam layout yang lebih rapi tanpa mengubah mesin perhitungannya.

Kelebihan OSRM + OpenStreetMap

Rute dihitung mengikuti jaringan jalan sebenarnya, sehingga hasilnya lebih representatif dibanding jarak garis lurus semata.

Input A Excel / CSV
Input B KML
Output Tabel + Peta
Nilai Utama

Kenapa workflow ini lebih presisi

  1. Rute jalan akurat karena memakai OSRM untuk menghitung jalur aktual di jaringan jalan.
  2. Database alat produksi dibaca dari KML sehingga hasil pencarian titik terdekat tetap konsisten.
  3. Perhitungan massal membantu menganalisis banyak lokasi sekaligus dalam satu proses.

Teknologi: OpenStreetMap untuk jaringan jalan dan OSRM untuk routing.

Langkah Kerja

Panduan penggunaan singkat

Anda bisa mengikuti alur berikut untuk memastikan input, proses, dan review hasil berjalan mulus.

1. Upload data

Unggah file Excel atau CSV untuk daftar lokasi A, lalu unggah file KML untuk database alat produksi lokasi B.

2. Proses perhitungan

Setelah kedua file valid, jalankan perhitungan dan pantau progress serta status routing OSRM.

3. Tinjau hasil

Periksa ringkasan metrik dan tabel hasil untuk melihat alat produksi terdekat, jarak lurus, dan jarak jalan.

4. Buka peta rute

Pilih tab peta, batasi jumlah rute yang ingin ditampilkan, lalu perbarui visualisasi jika perlu.

5. Unduh hasil

Gunakan tombol download Excel untuk menyimpan hasil analisis dalam format yang siap dibagikan atau diolah lanjutan.
