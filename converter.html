Format file yang didukung

CSV untuk konversi ke KML

File CSV minimal memiliki kolom berikut:

  • name: Nama lokasi atau titik
  • latitude: Koordinat lintang desimal
  • longitude: Koordinat bujur desimal
  • description: Deskripsi tambahan opsional

Contoh isi file CSV:

name,latitude,longitude,description
Tugu Monas,-6.175392,106.827153,Monumen Nasional
Bundaran HI,-6.194444,106.822956,Bundaran Hotel Indonesia

Struktur KML untuk konversi ke CSV

File KML sebaiknya berisi placemark dengan koordinat point yang valid.

<Placemark>
  <name>Tugu Monas</name>
  <description>Monumen Nasional</description>
  <Point>
    <coordinates>106.827153,-6.175392,0</coordinates>
  </Point>
</Placemark>
