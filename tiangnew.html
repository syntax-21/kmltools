Petunjuk

Alur penggunaan singkat

  1. Buat path di Google Earth dan simpan sebagai KML atau KMZ.
  2. Upload file tersebut ke tool ini atau drag and drop ke area upload.
  3. Tentukan jarak antar placemark dan nama placemark bila diperlukan.
  4. Generate file KML baru, preview hasil tabel, atau tampilkan hasil di peta.

Tutorial

Panduan penggunaan aplikasi

Gunakan panduan ini jika Anda ingin mengikuti proses pembuatan path hingga ekspor hasil secara visual.

  1. Buat path di Google Earth.
  2. Export path ke file KML atau KMZ.
  3. Upload file, atur interval, lalu generate hasil.
  4. Gunakan preview tabel dan peta sebelum file dipakai lebih lanjut.
