Cara tool membaca file existing

  1. Upload satu file KML atau KMZ yang sudah berisi path, tiang, dan objek lapangan lain.
  2. Tool akan membaca semua path dan titik tanpa membuat placemark tiang baru.
  3. Ikon titik di peta mengikuti style KML asli selama URL ikonnya bisa diakses browser.
  4. Titik referensi terdekat pada radius 250 meter dapat memicu perhitungan PU-AS-HL beserta seberangannya.
  5. Pada line lurus, PU-AS-DE-50/70 atau PU-AS-HL juga dihitung tiap 250 meter.
  6. PU-AS-SC sekarang dihitung dari tiang sebelum dan sesudah titik PU-AS-DE-50/70 atau PU-AS-HL yang terkait di path.
  7. Kategori tiang yang dikenali untuk aturan sebelum dan sesudah adalah TE (existing), TN (new), TB (baru), dan TP (plan).

Tutorial

Panduan penggunaan aplikasi

Gunakan panduan ini untuk menyiapkan file existing, memeriksa hasil preview, dan mengekspor KML asesoris.

  1. Siapkan satu file KML atau KMZ existing dari Google Earth.
  2. Upload file ke tool ini dan jalankan preview hasil.
  3. Periksa tabel dan peta untuk memastikan objek existing terbaca dengan benar.
  4. Unduh KML hasil yang menambahkan folder asesoris tiang.
