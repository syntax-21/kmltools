Tutorial dan catatan penting

Bagian ini merangkum langkah kerja dan tips agar file KML Anda lebih mudah diproses.

Langkah penggunaan

  1. Upload file KML yang berisi path LineString.
  2. Masukkan panjang segmen yang diinginkan dalam meter.
  3. Proses file dan tunggu ringkasan hasil muncul.
  4. Pilih unduhan gabungan atau per segmen sesuai kebutuhan.
  5. Buka peta jika Anda ingin memeriksa visualisasi segmen.

Catatan penting

  • Pastikan file KML berisi LineString yang valid.
  • Semakin kecil panjang segmen, semakin banyak file yang akan dihasilkan.
  • Path dengan titik yang cukup rapat biasanya memberi hasil pembagian yang lebih baik.
